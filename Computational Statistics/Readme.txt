This folder contains some of my work for the computational statistics course. The weeks cover following topics:

Week1 - Newton-Raphson method and its variants (Gauss-Newton, Steepest Descent, etc)
Week2 - Simulated annealing and genetic algorithms
Week3 - EM algorithm, inverse transform method and rejection sampling
Week4 - Monte carlo methods, MCMC techniques
Week5 - Penalized regression splines
Week6 - Bootstrap methods
